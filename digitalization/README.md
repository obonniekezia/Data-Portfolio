# Digitalization and Income Level Prediction

## Project Description

This project conducts a descriptive and predictive statistical analysis to study the relationship between digitalization and economic development across countries.

The first objective is to classify countries according to their income level using digitalization indicators and machine learning techniques.  
The second objective is to estimate and analyze the determinants of a digitalization index using macroeconomic variables.

## Data

The dataset includes country-level indicators related to:

- Digital infrastructure and connectivity
- Technology adoption
- Internet access and digital services
- Macroeconomic variables such as GDP, trade, and development indicators

## Methods

The analysis involves several steps:

- Data cleaning and preprocessing
- Construction of a digitalization index
- Exploratory data analysis
- Machine learning classification to predict income levels
- Regression analysis to identify macroeconomic determinants of digitalization

## Tools

- Python / R
- Machine learning models
- Statistical modeling
- Data visualization

## Output

The project provides:

- A predictive model for classifying countries by income level
- An econometric analysis of the relationship between digitalization and macroeconomic variables
- Insights into the role of digitalization in economic development
