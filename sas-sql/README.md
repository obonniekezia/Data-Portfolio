# SQL Analysis – Toulouse Construction Projects (2024)

## Project Description

This project performs a SQL-based analysis of construction projects in Toulouse in 2024. The objective is to structure and analyze a database containing information on construction sites in the city.

The project involves the creation of database tables and the use of SQL queries to explore and analyze the data.

## Data

The dataset contains information on construction projects in Toulouse in 2024, including variables related to:

- Project location
- Type of construction work
- Project duration
- Project characteristics and administrative information

## Methods

The analysis includes several steps:

- Creation of database tables
- Data structuring and organization
- SQL queries to explore the dataset
- Aggregation and descriptive analysis of construction projects
- Identification of patterns and key characteristics of construction activity in Toulouse

## Tools

- SAS
- SQL
- Data querying and database management

## Output

The project produces a structured database and a set of SQL queries allowing an exploratory analysis of construction activity in Toulouse in 2024.
